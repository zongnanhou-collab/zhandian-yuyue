# 针绽预约 - H5网页版

## 项目结构

```
web/
├── index.html      # 主页面
├── css/
│   └── style.css   # 中国风样式
└── js/
    └── app.js      # 业务逻辑
```

## 功能特点

✅ 患者端
- 14天内预约选择
- 显示每日可预约时间段
- 6种治疗项目选择
- 支持隐藏姓氏（隐私保护）
- 查看和取消自己的预约

✅ 医师端
- 预约管理（日视图）
- 权限设置（周末开关、微信绑定）
- 数据统计

## 使用方式

### 方式1：本地预览
直接用浏览器打开 `web/index.html` 即可预览

### 方式2：部署上线（推荐）

**方案A：Vercel（免费）**
1. 安装 Vercel CLI：`npm i -g vercel`
2. 进入 web 目录：`cd web`
3. 部署：`vercel --prod`

**方案B：GitHub Pages**
1. 将 web 文件夹内容推送到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages

**方案C：自己的服务器**
- 将文件上传到你的服务器即可

## LeanCloud 配置

当前版本使用本地模拟数据。要启用真正的数据存储：

1. 注册 [LeanCloud](https://leancloud.cn/)
2. 创建应用
3. 在 `js/app.js` 中替换：
   ```javascript
   appId: 'your-app-id',
   appKey: 'your-app-key'
   ```
4. 取消代码中 LeanCloud 相关注释
